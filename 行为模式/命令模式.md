## 四、代码实现
1、抽象命令类(Command)
抽象命令类一般是一个接口，在其中声明了用于执行请求的execute()等方法，通过这些方法可以调用请求接收者的相关操作。
```java
package com.atmae.command;

/**
 * @Author: Mae
 * @Date: 2022/5/26
 * @Time: 9:33
 * @Description:
 */
public interface Command {
    void execute();
}

```
